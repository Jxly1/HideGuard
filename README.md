# 🛡️ HideGuard – Hide Windows on Discord Streams

**HideGuard** is a lightweight and easy-to-use tool that allows you to hide specific windows or applications while streaming on Discord.

Whether you're streaming to friends or a larger audience, StreamGuard helps you protect your privacy by making selected programs completely invisible on your stream — while they remain fully visible and usable on your own screen.

---

## 🚀 Features

* 🎯 Hide any selected window or application
* 🖥️ Works in real-time while streaming
* 👁️ Invisible to viewers, visible to you
* ⚡ Simple and fast user interface
* 🔒 Perfect for privacy while streaming
* 🎮 Optimized for Discord streaming

---

## 📥 Download

1. Go to the **Releases** section of this repository
2. Download the latest version (`.exe` file)
3. No installation required

---

## ▶️ How to Use

Using HideGuard is extremely simple:

1. Open the downloaded `.exe` file
2. The program will display a list of currently running applications
3. Select the window or program you want to hide
4. Click the **Hide** button

✅ That’s it!

The selected window will now:

* ❌ NOT be visible on your Discord stream
* ✅ Still be visible and usable on your own screen

---

## 💡 Example Use Cases

* Hide Discord, browser tabs, or private chats
* Keep tools or scripts hidden while streaming
* Protect sensitive information
* Clean and professional streams

---

## ⚠️ Important Notes

* HideGuard only affects what viewers see on your stream
* It does NOT close or modify the selected program
* Some applications (like games with anti-cheat) may not be supported
* Make sure HideGuard is running before or during your stream
